# 프레임워크 스터디

## 스프링

본 레포지토리는 프레임워크 스터디 중 스프링 프레임워크를 학습하는 프로젝트를 저장합니다.

--- 

### 개요

유저 정보를 저장하고 로그인을 하여 jwt인증을 하는 간단한 rest api 서버를 만듭니다.

### 환경

| 구분      | 내용              |
|---------|-----------------|
| 데이터 베이스 | MySQL           |
| ORM     | Spring Data JPA |
| 인증 방식   | JWT             |
| 삭제 방식   | Soft Delete     |
| 기타      | 롬복 사용           |

### 구현할 내용

- user crud
- 회원가입, 탈퇴, 로그인 로직을 처리하는 security
- 공통 예외 필터, 핸들러 등을 관리하는 common
- 커스텀 예외를 이용한 예외 처리
- dto와 entity를 매핑해주는 mapper
- jwt 인증
